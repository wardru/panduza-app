# Panduza app

## Prerequisites

Follow Tauri's Prerequisites page and install all dependencies for your platform including NodeJS (we use a javascript frontend)

## Install deps

```
npm install --legacy-peer-deps
```

## Build (dev mode)

```
npm run tauri dev
```

## Build (release mode)

```
npm run tauri build
```

## JS Lint

```
npm run lint
```

## Prettier

```
npm run format
```
